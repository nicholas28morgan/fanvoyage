# Deploying FanVoyage Beta v3 to Vercel (quick)

1. Create a GitHub repo called `fanvoyage` under your account (e.g. nicholas28morgan/fanvoyage).
2. Push this project to that repo.
3. In Vercel, Import Project → select your GitHub repo → Framework: Next.js.
4. Add env vars in Vercel settings (EXPEDIA_KEY, EXPEDIA_SECRET, TICKETMASTER_API_KEY, SG_CLIENT_ID, etc.).
5. Deploy and test the live URL.
